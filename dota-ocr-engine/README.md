# DotA OCR Engine

Parses DotA 2 screenshots to scrap data for EloElo.


# Dependencies

## Linux (Ubuntu)

The new and wonderful python package manager:
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

OCR libraries
```
sudo apt install tesseract-ocr libtesseract-dev
```

## Windows

Not supported.
